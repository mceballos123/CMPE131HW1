JavaScript Amazon Clone Project

Project Description

This project is an Amazon clone website built using HTML, CSS, and JavaScript. It mimics the functionality of an online shopping platform, allowing users to interact with various features

Features

  • Home Page
      • -Users can browse and select items from the displayed product listings
  • Quantity Selector
      • -Users can choose the quantity of each item
  • Checkout Page
      • -Users can update items quantities and select their preferred delivery options
      Shows real time selection for the various delivery options
  • Order Summary
      • -Displays shipping and handling costs
      -total cost before and after taxes
      Total number of items in the cart

Technologies Used

The project was build using:

  • HTML for structuring the home and checkout page
  • CSS for styling both pages
  • JavaScript for handling the logic and making it interactive
